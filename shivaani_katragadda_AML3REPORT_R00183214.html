The given dataset is in the form of images, so for pre-processing we have to follow some concepts to preparing the data. Here I am performing the following tasks to prepare the data. They are as follows:

  1. Loading the data
  2. Resizing the image
  3. Normalizing the image
  4. Flattening the image
  5. Dimension Reduction

1.Loading the dataset:¶

There are so many packages available in python to read the images data such as

  1. Opencv
  2. PIL
  3. Pyplot
  4. matplotlib.image

Here I will be using the Opencv python package for loading the data. In the OpenCV package there is one function cv2.imread function is available by using this function I read the images data.

2.Resizing the image:¶

An image is resized to increases or decreases the width and height of the image in pixels. This process is also called resampling. Here I will be using 350 350 3 sizes for resizing the images, Where IMAGE_WIDTH=350, IMAGE_HEIGHT=350,IMAGE_SIZE=(IMAGE_WIDTH, IMAGE_HEIGHT),IMAGE_CHANNELS=3. I did resizing by using the function cv.resize() which is available in Opencv package.

3.Normalizing the image:¶

Image normalization is a common process that adjusts the range of pixel intensity values in image processing. Its normal purpose is to convert an input image into a series of more familiar pixel values. This process is called Normalization. Here I used cv.normalize() to normalize the picture

4.Flattening the image¶

Flattening occurs when all the Photoshop layers are reduced to one background layer. Layers will increase the size of files, while also tying up scarce resources for processing. You can choose to combine some layers or even flatten the entire image to one background layer to keep the file size down. Here I flattened the image by using flatten() function

5.Dimension Reduction¶

Reduction of dimensionality is the method of raising the number of random variables under consideration, by acquiring a collection of key variables. It can be divided into the collection of functions and the extraction of features.

The various methods used for dimensionality reduction include:

Principal Component Analysis (PCA) Linear Discriminant Analysis (LDA) Generalized Discriminant Analysis (GDA) Dimensionality reduction may be both linear or non-linear, depending upon the method used.

Here I am using PCA METHOD for reducing the data.

Principal Component Analysis

Karl Pearson introduced the form. It operates on a condition that when the data is converted to data in a lower dimension space in a higher-dimensional space, the variance of the data in the lower dimensional space should be maximum.

I used PCA() function which is available in sklearn.decomposition.For the n_components I have assigned the value 25.

Logistic regression is a classification algorithm used to assign a distinct set of groups to observations. Logistic regression transforms its output by returning a probability value increasing the logistic sigmoid function.

A Naive Bayes Classifier is a supervised machine-learning algorithm using the theorem of the Bayes that assumes features are statistically independent. It has proved itself to be a classifier with strong results regardless of the conclusion.

K nearest neighbors is a simple algorithm which stores all available cases and classifies new cases based on a measure of similarity

"Support Vector Machine" (SVM) is a supervised algorithm for machine learning, that can be used for classification or regression challenges. It's mainly used in classification issues . Support Vectors are essentially individual analytical co-ordinates.

The decision tree builds a tree structure in the form of classification or regression models. This breaks down a collection of data into smaller and smaller subsets while at the same time incrementally creating a related decision tree. A node of decision has two branches or more. The node to the leaf is a classification or decision.

The random forest is a classification algorithm made up of lots of decision trees. When constructing each individual tree it uses bagging and features randomness to try to establish an uncorrelated tree forest whose prediction by committee is more accurate than that of any individual tree.

SGDClassifier is an SGD training Linear classifier, the Stochastic Descent of the Gradient (sgd) is a solver. This is an easy and effective approach to the discriminative learning of linear classifiers under convex loss functions such as (linear) supporting vector machines and logistic regression

Gradient boosting is a machine learning technique for regression and classification problems that generates a predictive model in the form of an ensemble of poor predictive models, usual trees of decisions.

An AdaBoost classifier is a meta-estimator that starts by fitting a classifier on the original dataset and then fits additional classifier copies on the same dataset but adjusts the weights of incorrectly classified instances so that subsequent classifiers are more focused on difficult cases.

I have used above mentioned nine classifier algorithms for building the models.

def plot_confusion_matrices(models, confusion_matrices):
    fig, axs = plt.subplots(2,4, figsize=(10,5)) 

    m = 0
    for i, ax_r in enumerate(axs):
        for j, ax in enumerate(ax_r):
            sns.heatmap(confusion_matrices[m], annot=True, cbar=False, cmap='Blues', fmt='g', ax = ax)
            ax.set_xlabel('Predicted label')
            ax.set_ylabel('True label')
            ax.set_title(f'{list(models.keys())[m]}', fontsize=12, fontweight='bold')
            m += 1

    plt.subplots_adjust(top=0.92, bottom=0.08, left=0.10, right=0.95, hspace=0.25,
                        wspace=0.35)
    plt.tight_layout()
    plt.show()
